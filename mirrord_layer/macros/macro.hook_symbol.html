Macro mirrord_layer::macros::hook_symbol

source · 
macro_rules! hook_symbol {
    ($hook_manager:expr, $func:expr, $detour_name:expr) => { ... };
}
Expand description

Used by go_hooks to hook go syscalls with HookManager::hook_symbol_main_module.

Parameters

  • $hook_manager: a valid HookManager instance that is used to hook the symbol $func;

  • $func: the function we want to replace;

  • $detour_name: detour we’re replacing $func with.

Examples

  • Hooking go’s syscall.RawSyscall.abi0:
hook_symbol!(
    hook_manager,
    "syscall.RawSyscall.abi0",
    go_rawsyscall_detour
);
