OPEN_FILES is used to track open files and their corrospending remote file descriptor. We use Arc so we can support dup more nicely, this means that if user Opens file A, receives fd 1, then dups, receives 2 - both stay open, until both are closed. Previously in such scenario we would close the remote, causing issues.
