Structs

ConnectResult 🔒
Helper struct for connect results where we want to hold the original errno when result is -1 (error) because sometimes it’s not a real error (EINPROGRESS/EINTR) and the caller should have the original value. In order to use this struct correctly, after calling connect convert the return value using this .into/.from then the detour would call .into on the struct to convert to i32 and would set the according errno.
