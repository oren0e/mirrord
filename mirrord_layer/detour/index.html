The layer uses features from this module to check if it should bypass one of its hooks, and call the original libc function.

Here we also have the convenient Detour, that is used by the hooks to either return a Result-like value, or the special Bypass case, which makes the detour function call the original libc equivalent, stored in a HookFn.
